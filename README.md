##  Cómo ejecutar el proyecto con Docker

### 1. Requisitos:

- Tener Docker Desktop instalado
- Tener MongoDB Atlas activo y el `MONGO_URL` de tu base

---

### 2. Cloná este repositorio:

git clone https://github.com/alexiscorrea14/backend-3.git
cd backend-3
